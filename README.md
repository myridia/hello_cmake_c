Example how to use Cmake 

## Install
```
git clone https://github.com/myridia/hello_cmake_c.git
cd hello_cmake_c
cmake -B build 
cmake --build build
```

## Run the binary
```
./build/Debug/hello_cmake_c
```

## For convenience with Linux, you can run the shell file run.sh
```
./run.sh
```
